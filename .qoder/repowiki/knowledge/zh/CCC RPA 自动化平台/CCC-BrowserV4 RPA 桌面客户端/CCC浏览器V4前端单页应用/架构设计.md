- **核心架构**：采用 Vue 3 + Vite + TypeScript 技术栈，通过 `main.ts` 统一初始化 Pinia 状态管理、Vue Router 路由及 Element Plus UI 库。
- **跨模块通信**：`router/index.ts` 集成全局前置守卫，依赖 `auth` 模块的 Store 实现路由级身份鉴权；`utils/tauri-bridge.ts` 封装 Tauri `invoke` 调用，为 `auth`（登录回调）和 `core`（设备标识）模块提供原生能力桥接。
- **数据流协同**：`api/request.ts` 定义统一的 Axios 实例，供 `auth`、`tasks` 及 `execution` 模块复用，确保 API 请求路径与拦截逻辑的一致性。
- **布局编排**：`AppLayout.vue` 作为受保护路由的根组件，动态加载 `SideMenu` 与 `StatusBar`，并嵌套渲染业务页面，实现导航与业务逻辑的视觉隔离。

- **框架核心**: 采用 `FastAPI` (v0.115.0) 作为 Web 框架，`Uvicorn` 作为 ASGI 服务器。
- **配置管理**: 使用 `pydantic-settings` 实现类型安全的配置加载，支持通过 `.env` 文件动态切换 MySQL 或 SQLite 数据库后端 (`app/config.py`)。
- **数据持久层**: 基于 `SQLAlchemy` (v2.0) 构建 ORM 层，在 `app/database.py` 中统一管理引擎创建与会话工厂 (`SessionLocal`)，并提供用于依赖注入的 `get_db` 生成器。
- **模型规范**: 定义 `Base` 声明式基类与 `TimestampMixin` 混入类，为所有数据模型提供统一的 `created_at` 和 `updated_at` 审计字段 (`app/models/base.py`)。
- **接口路由**: API 路由按功能模块划分（如 `app/api/health.py`），通过 `APIRouter` 组织并挂载，目前实现了包含数据库状态检测的健康检查端点。

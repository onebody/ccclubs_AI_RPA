- **混合架构编排**：采用 Tauri v2 作为宿主容器，前端（Vue3）负责 UI 渲染与状态管理，后端（FastAPI）提供独立业务 API，Tauri Rust 层作为中间件处理系统级调用（如设备 ID、本地回调服务器）。
- **跨层通信契约**：前端通过 `tauri-bridge.ts` 封装的 `invoke` 命令与 Rust 层交互，获取持久化设备标识及登录回调端口；Rust 层通过 `tauri_plugin_opener` 唤起外部浏览器完成 OAuth/SSO 流程。
- **基础设施隔离**：通过 `docker-compose.yml` 独立部署 MySQL 数据库，后端 API 与桌面客户端共享同一数据源，实现云端业务逻辑与本地客户端能力的解耦。

### 租户 API 组件
**更新** 租户 API 已完全重构为数据库驱动实现，提供完整的 CRUD 操作：

- **列表查询**：获取所有活跃租户，返回简化格式供前端下拉选择
- **详情获取**：根据租户ID获取完整租户信息
- **创建租户**：验证租户编码唯一性，创建新租户并返回详细信息
- **更新租户**：支持部分字段更新，包括名称、省份和激活状态
- **软删除**：将租户的 is_active 字段设为 False 实现逻辑删除
